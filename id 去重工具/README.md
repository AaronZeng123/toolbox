# ID Deduplication Tool

## 项目描述
这是一个简单的 HTML 工具，用于对 ID 列表进行去重。

## 功能列表

### ID 去重工具
- **功能描述**: 用户输入包含重复 ID 的文本（通常由逗号分隔），点击按钮后工具会自动去除重复项。
- **实现方案**:
  - 使用 HTML 创建用户界面，包含输入框、操作按钮和结果展示区域。
  - 使用 JavaScript 处理字符串：
    - 将输入字符串按逗号分隔成数组。
    - 使用 `Set` 数据结构进行去重。
    - 统计去重前后的数量并显示。
    - 提供一键复制功能，将去重后的结果复制到剪贴板。
    - 复制成功后显示 Toast 提示。

### 结果展示格式切换
- **功能描述**: 用户可以选择去重结果的展示方式：
  - **逗号分隔**: 默认方式，适合一般文本使用。
  - **换行排列**: 每个 ID 占一行，方便直接复制粘贴到 Excel 表格中。
- **实现方案**:
  - 在结果区域上方增加单选按钮（Radio Buttons）供用户切换格式。
  - 监听单选按钮的变化，实时更新结果文本框的内容。
  - 内部保存去重后的 ID 数组，根据当前选择的格式进行 `join(',')` 或 `join('\n')`。
