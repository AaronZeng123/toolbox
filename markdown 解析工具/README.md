# Markdown 解析工具

这是一个基于 Web 的 Markdown 解析和预览工具。它允许用户输入 Markdown 文本或上传 Markdown 文件，并实时预览生成的 HTML。

## 功能

- **实时预览**: 在输入 Markdown 文本时即时查看 HTML 渲染结果。
- **文件支持**: 支持上传 `.md` 或 `.txt` 文件进行解析。
- **批量处理**: 支持一次上传多个文件。
- **导出**:
  - 下载 HTML 文件。
- **样式增强**:
  - 自动对标题（H1-H6）应用不同颜色，增强可读性。
  - 加粗文本显示为红色，突出重点。
  - **优化布局**: 调整了左右分栏比例（左侧输入框较窄，右侧预览框较宽），并增加了垂直高度，提升阅读体验。

## 技术栈

- **HTML/CSS/JS**: 原生 Web 技术构建。
- **Marked**: 用于将 Markdown 转换为 HTML。
- **DOMPurify**: 用于净化 HTML，防止 XSS 攻击。
- **Sakura CSS**: 提供开箱即用的美观样式。

## 使用说明

1. 直接在左侧文本框粘贴 Markdown 内容。
2. 或者点击 "选择文件" 按钮上传本地 Markdown 文件。
3. 右侧将显示渲染后的 HTML 预览。
4. 点击 "下载HTML" 按钮保存结果。
