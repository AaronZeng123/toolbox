# 哈希 256 加密工具

这是一个简单的 HTML 工具，用于将输入文本进行 SHA-256 加密，并自动提取加密结果的前 6 位字符。

## 功能列表

- **文本输入**: 提供一个文本域用于输入需要加密的内容。
    - **默认填充**: 页面加载时，自动以当前日期（YYYYMMDD格式，如20251207）作为默认内容，方便日常使用。
    - **手动修改**: 用户可以随意修改输入内容。
- **实时加密**: 输入内容时自动计算 SHA-256 哈希值。
- **结果展示**:
    - **前 6 位**: 醒目展示哈希值的前 6 位字符，方便查看。
    - **完整哈希**: 展示完整的 64 位十六进制哈希字符串。
- **一键复制**: 提供"复制"按钮，可快速将前 6 位哈希值复制到剪贴板。

## 实现方案

- **核心算法**: 使用浏览器原生 Web Crypto API (`crypto.subtle.digest('SHA-256', ...)`) 进行高性能、安全的本地加密计算，无需将数据发送到服务器。
- **界面交互**:
    - 使用原生 HTML/CSS/JavaScript 构建，无外部依赖。
    - **初始化逻辑**: 页面加载时执行 `initializeWithDate` 函数，获取当前时间并格式化为 `YYYYMMDD` 填充至输入框，随后立即触发一次哈希计算。
    - 监听 `textarea` 的 `input` 事件实现实时计算。
    - 兼容性处理：复制功能优先使用 `navigator.clipboard` API，并提供 `document.execCommand('copy')` 作为回退方案，确保在不同浏览器环境中均可使用。

## 使用方法

1. 用浏览器打开 `index.html` 文件。
2. 在输入框中输入任意文本。
3. 查看下方显示的"前 6 位结果"和"完整哈希值"。
4. 点击"复制"按钮即可获取前 6 位哈希值。
