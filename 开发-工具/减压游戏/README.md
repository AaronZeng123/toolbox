# 解压键盘侠 (Stress Buster Keyboard)

这是一个通过电脑键盘直接玩的极度减压网页游戏。

## 功能描述
- **键盘交互**: 按下键盘任意键即可触发炫酷的视觉特效和音效。
- **解压文案**: 每次敲击随机弹出“滚”、“加薪”、“暴富”、“退退退”等极度解压的中文词汇。
- **触屏支持**: 支持移动端触屏点击，随时随地解压。
- **连击系统**: 快速敲击触发 COMBO 连击计数，字体随连击数动态放大。
- **视觉特效**: 包含多彩粒子爆炸、屏幕震动以及按键字符的动态浮现。
- **动态音效**: 使用 Web Audio API 实时合成复古街机风格的音效，无需加载外部音频文件。
- **极简运行**: 单个 HTML 文件，双击浏览器打开即可游玩。

## 实现方案
- **HTML5 Canvas**: 用于渲染高性能的粒子系统和动画效果。
- **Web Audio API**: 用于生成每一次按键的独特音效（频率随机化）。
- **原生 JavaScript**: 无第三方依赖，保证极速加载。

## 使用说明
1. 下载 `game.html` 文件。
2. 双击使用 Chrome、Edge 或 Safari 等现代浏览器打开。
3. 戴上耳机，开始疯狂敲击键盘！

## 版本记录

### Bug 修复 (2025-12-11)
- **问题描述**: 当连击数 (COMBO) 过大时，计数文本会放大并超出屏幕右侧，导致无法看全数字。
- **修复方案**: 
    1. 将计数器的 CSS `right` 属性从 `20px` 调整为 `60px`，预留更多空间。
    2. 新增 `transform-origin: right top;` 属性，确保文本放大时向左下方扩展，而不会溢出屏幕右边界。
